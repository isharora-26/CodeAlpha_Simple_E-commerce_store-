# 🛒 Simple Ecommerce Store

A Django-based Simple Ecommerce Store developed as part of the **CodeAlpha Python Development Internship**.

## Features

- User Registration & Login
- Product Listing
- Product Details
- Add to Cart
- Update Cart Quantity
- Remove from Cart
- Checkout
- Order Confirmation
- Order History
- Django Admin Panel

## Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite

## Project Structure

```
CodeAlpha_Simple_E-commerce_store-
│
├── SimpleEcommerceStore/
│   ├── ecommerce/
│   ├── store/
│   ├── templates/
│   ├── static/
│   ├── manage.py
│   └── README.md
│
└── requirements.txt
```

## How to Run

```bash
cd SimpleEcommerceStore
pip install -r ../requirements.txt
python manage.py migrate
python manage.py runserver
```
  
Visit:

http://127.0.0.1:8000/
