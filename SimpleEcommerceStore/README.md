# Simple Ecommerce Store

A Django-based Simple Ecommerce Store developed as part of the CodeAlpha Python Development Internship.

## Features

- User Registration & Login
- Product Listing
- Product Details
- Add to Cart
- Update Cart Quantity
- Remove from Cart
- Checkout
- Order Confirmation
- Order History
- Admin Panel for Product Management

## Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite

## How to Run

1. Clone the repository

git clone <repository-url>

2. Install dependencies

pip install -r requirements.txt

3. Run migrations

python manage.py migrate

4. Start the server

python manage.py runserver

5. Open

http://127.0.0.1:8000/
